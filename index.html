A Bit About Me...

Hello, my name is Borislav. Presently I work as a system administrator and a freelance web developer.

Due to my passion for computer technology, I found out I really enjoy coding. This website is to showcase some of my projects and to help me pick up some side work.

Here are some of the technologies I use:

  • HTML5, CSS, JavaScript ES6
  • SASS, JSX, jQuery, React
  • Bootstrap, MaterializeCSS
  • NodeJS, Express
  • SQL, MongoDB

I graduated the Technical University of Plovdiv, as an engineer in Computer Systems and Technology.

After which I started working as an RTL Verification specialist with SystemVerilog and UVM. That position helped me further my knowledge in object oriented programming, use of standardized methodologies, version control solutions and automated bugtesting.

Eventually I moved to system administration, which helped me improve my knowledge in virtualization, networking, computer automation and working with databases. Now, I seek to expand my knowledge into the vast realm of full stack web development.

Some of my work:

Simon Says

Simon Says

A memory game made for FreeCodeCamp. Users are required to remember a sequence of steps, up to 20 levels. Game features a strict mode that resets the progress whenever a mistake is made.

Calculator

Javascript Calculator

Calculator written for FreeCodeCamp. With this project I did not only want to emulate a physical calculator, but make it so the user is able to do fast, seamless calculations with keyboard inputs.

Pomodoro Clock

Pomodoro Clock

A productivity timer app made for FCC. The idea is for the user to have a working session followed by a break session. The user can change the length of the sessions or pause. The sessions duration are remembered in the browser's local storage.

Tic Tac Toe

Tic Tac Toe

A Tic Tac Toe game made for FFC. The user can select to play as either X and start first, or O to start second.

Twitch Watcher

Twitch JSON API App

Another app made for FCC, where the user can check if their favorite streamers are currently streaming on twitch. It uses the twitch API. While the app offers a pre-made list of streamers, the user is able to customize their own, stored in the browser's local storage.

Weather App

Weather App

My first project using an external API. It is a weather app made for FCC. After being granted access to the user's location, it gives a weather forecast at the given location.

Random Quote Generator

Random Quote Generator

A quote generator app for FCC. The user can generate random quotes and post them on twitter.

Wikipedia Finder

Wikipedia Finder

A Wikipedia search app using the Wikipedia API to display a list of results with links to the pages. There is also a button that opens up a random article. Made as part of the FCC certification process.
